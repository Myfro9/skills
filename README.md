# Codex Skills

我的个人 Codex Skills 集合。

## 什么是 Codex Skills?

Codex Skills 是用于帮助 Codex CLI 完成特定任务的可复用能力包。每个 skill 包含：
- `SKILL.md` - Skill 的描述和使用说明
- `agents/` - Agent 配置文件
- `scripts/` - 辅助脚本
- `assets/` - 资源文件

## 技能列表

| Skill | 描述 |
|-------|------|
| `algorithmic-art` | 生成算法艺术图案 |
| `brand-guidelines` | 品牌指南相关任务 |
| `canvas-design` | Canvas 设计工具 |
| `claude-api` | Claude API 集成 |
| `cubemx-prj-create` | STM32CubeMX 项目创建 |
| `doc-coauthoring` | 文档协作 |
| `docx` | Word 文档处理 |
| `frontend-design` | 前端设计 |
| `mcp-builder` | MCP 服务器构建 |
| `pdf` | PDF 文件处理 |
| `pptx` | PowerPoint 处理 |
| `skill-creator` | Skill 创建工具 |
| `vivado-prj-create` | Vivado 项目创建 |
| `web-artifacts-builder` | Web 构建工具 |
| `webapp-testing` | Web 应用测试 |
| `xlsx` | Excel 文件处理 |

## 安装使用

```bash
# 克隆仓库
git clone https://github.com/myfro9/codex-skills.git ~/.codex/skills

# 或者同步现有 skills
git pull origin main
```

## 创建新 Skill

参考 `skill-creator` skill 来创建新的技能。
